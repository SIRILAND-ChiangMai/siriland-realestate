SIRILAND Professional 2026 - GitHub Upload

1) Open admin.html from the GitHub Pages site or local server.
2) Add/Edit property.
3) Select photos.
4) Save Property.
5) Download ZIP.
6) Open the ZIP.
7) Copy properties.js into the main GitHub project folder.
8) Copy everything inside ZIP/images into the project images folder.
9) Commit + Push.
10) Open the live site and hard refresh.

Important:
- Do not rename image files after export.
- If a property has no photos, add photos before publishing.
- GitHub Pages uses properties.js, not data/properties.json.
