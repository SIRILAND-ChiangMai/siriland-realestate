# SIRILAND CMS PRO Sprint 3.1

## Summary
Sprint 3.1 adds a professional property detail experience with direct property URLs, sharing tools, QR code, print support, favorites, Google Maps actions, and a first version of the Ask About This Property lead form.

## New Features
- Direct Property URL: `?property=PROPERTY_ID`
- Share Center: Facebook, LINE, WhatsApp, Copy Link
- QR Code for each property
- Print-friendly property page
- Favorite button using localStorage
- Google Maps / Street View button
- Ask About This Property form
- Save Lead locally in browser storage
- Send inquiry via LINE or WhatsApp with property ID and link

## Updated Files
- `script.js`
- `style.css`

## Notes
This sprint does not modify the admin panel or property data. It is built on the real uploaded SIRILAND project files.
