SIRILAND Professional Final

Upload method:
1. Extract this ZIP.
2. Copy all files into your GitHub Desktop repository folder.
3. Replace existing files.
4. Commit to main.
5. Push origin.

https://siriland-chiangmai.github.io/siriland-realestate-v2/

Admin CMS:
https://siriland-chiangmai.github.io/siriland-realestate-v2/admin.html

How to add property:
1. Open admin.html.
2. Click Add Property.
3. Paste Facebook post text.
4. Click Auto Fill From Text.
5. Check details.
6. Select photos.
7. Save Property.
8. Download ZIP.
9. Upload generated properties.js and images to GitHub Desktop.
