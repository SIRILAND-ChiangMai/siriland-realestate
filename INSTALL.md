# SIRILAND Build 2026.07.10.04 — Full Multi-Language UI

1. ZIP dosyasını çıkarın.
2. Projedeki `index.html`, `script.js`, `style.css` dosyalarını bu pakettekilerle değiştirin.
3. `properties.js`, `admin.html` ve `images` klasörüne dokunmayın.
4. GitHub Desktop: Commit → Push origin.
5. Siteyi Ctrl+F5 ile yenileyin.

Not: Bu sürüm tüm sabit arayüz, filtre, ana sayfa koleksiyonları, detay butonları ve yapılandırılmış ilan alanlarını TH/EN/TR/ZH olarak çevirir. İlan açıklamalarının tam doğal çevirisi için admin içinde her dilin ayrı içerik alanı doldurulmalıdır.
