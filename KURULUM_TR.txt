SIRILAND PUBLISH MANAGER PRO v3

KURULUM
1. Bu ZIP'i şu klasöre çıkar:
   E:\SIRILAND_2030_Harddisk_Structure\01_CMS\Publish_Manager_v3

2. RUN_PUBLISH_V3.bat dosyasına çift tıkla.

3. Yollar hazır gelir:
   CMS Source:
   E:\SIRILAND_2030_Harddisk_Structure\01_CMS

   GitHub Repo:
   E:\SIRILAND_2030_Harddisk_Structure\09_GitHub\siriland-realestate

   Backup:
   E:\SIRILAND_2030_Harddisk_Structure\02_Backup

4. Önce "Analiz Et".
5. Hata yoksa "Publish".
6. GitHub Desktop'ta:
   Commit to main
   Push origin

YENİ ÖZELLİKLER
- Sadece yeni ve değişen dosyaları senkronlar.
- images klasörü dahil tüm değişiklikleri otomatik bulur.
- Sadece üzerine yazılacak eski dosyaları incremental yedekler.
- Duplicate ID, eksik ID, başlık, fiyat ve şehir kodunu kontrol eder.
- Eksik map ve görselleri uyarı olarak gösterir.
- GitHub klasöründe olup CMS'de olmayan orphan dosyaları raporlar; güvenlik için otomatik silmez.
- Commit mesajını panoya kopyalar.
- GitHub Desktop'u açmayı dener.
- Son 100 publish kaydını Publish History içinde tutar.
- Ayrıntılı publish raporu oluşturur.

ÖNEMLİ
- "Uyarılarda da publish'i durdur" seçili değilse eksik map/fotoğraf uyarıları publish'i engellemez.
- Gerçek validator hataları publish'i daima durdurur.
- Orphan dosyalar otomatik silinmez. Yanlışlıkla ilan veya fotoğraf kaybını önler.
